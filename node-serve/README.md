# Nodejs server

- 服务端渲染
> 查看页面源码可以看见  
> 比如电商网站商品列表展示,以及分页点击时页面会重新刷新  
> 好处可以是利于seo优化,便于用户搜索出来

- 客户端渲染
> 查看页面源码不可见  
> 比如电商网站的评论分页，点击时页面不用刷新，即可更新页面数据  
> 不利于seo收录  
> 所以不必要考虑搜索相关，采用ajax客户端渲染，关注用户体验


## 服务端控制可访问与否
- 并不是服务端所有的东西都可以请求到
- 和本地Apache不一样，本地Apache开启后www目录下都可以访问到

}
